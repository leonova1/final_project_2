Леонова Анна, проект 2
Дан файл с данными recipes_100.csv, содержащий данные о различных рецептах.
Файл answeres.json содержит ответы на задания:
	1. Все рецепты, в которых используется ингредиент (chicken),
	2. Три самых долгих для приготовления рецепта,
	3. Выведение названия блюд, которые можно приготовить на каждое количество человек. 
В файле Rating_of_dishes.png находится гистограмма, которая демонстрирует количество блюд с определенным рейтингом. 
В файле final_project_Leonova.py исходный код.
